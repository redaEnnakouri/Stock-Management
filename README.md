# Stock-Management

The objective of this Project is to generate  Stock Management 
* Items / purchases,
* Sales
* Stock
* Client
* Provider
* Sales Management
* Purchases Management
* Invoice
* Sales Report
* Customer Status
* Notes

 used PHP native 5 with bootstrap  and javascript
